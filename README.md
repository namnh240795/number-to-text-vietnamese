

### Installation
  yarn add number-to-text-vietnamese
  npm install number-to-text-vietnamese

### How to use
Support integer to text only
```
  import { getText } from 'number-to-text-vietnamese';
  getText(123456);
```

Support integer to text only with seperator
```
  import { getText } from 'number-to-text-vietnamese';
  getText(123456, '|');
```
